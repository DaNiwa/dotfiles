# Configuration Files

## Install

1. Clone the repository

```
git clone --recursive https://github.siri.apple.com/daniel-pham2/dotfiles.git ~/.dotfiles
```

2. Symlink the files

```
./install.sh
```

3. Install brew apps and OS X settings.

```
./brew.sh && ./osx.sh
```

4. Open vim and download plugins

```
:PlugInstall
```
